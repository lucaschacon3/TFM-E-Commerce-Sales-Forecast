# TFM---E-Commerce-Sales-Forecast
Proyecto de preprocesado, regresión para previsión de ventas y clustering para segmentación de clientes con el dataset E-Commerce Sales Forecast de Kaggle. El flujo incluyó búsqueda, prueba y selección de modelos óptimos para predecir demanda y agrupar perfiles de consumo, optimizando la toma de decisiones mediante análisis avanzado de datos.
